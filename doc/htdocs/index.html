¿Qué es LibWiiEsp?


Es una biblioteca libre de desarrollo de videojuegos en dos dimensiones para Nintendo Wii. Incluye la propia herramienta, un manual completo de instalación y uso, y el código fuente de tres juegos que ilustran cómo trabaja libWiiEsp.

La biblioteca está escrita en el lenguaje de programación C++, y tiene licencia GPLv3. Sus principales características son las siguientes:

  • Proporciona control sobre distintos sistemas de la videoconsola: vídeo, audio, almacenamiento y mandos (Wii Remotes).
  • Permite organizar y trabajar fácilmente con recursos multimedia: imágenes, efectos de sonido, pistas de música y fuentes de texto.
  • Incorpora un sistema de detección de colisiones basado en figuras geométricas simples.
  • Soporte de internacionalización, pudiéndose utilizar cualquier juego de carácteres Unicode.
  • Permite reproducir animaciones en dos dimensiones a partir de una imagen.
  • Incluye un sistema de registro de mensajes del sistema, para facilitar la depuración.
  • Sistema de plantillas (clases abstractas) para actores y escenarios del juego, pudiéndose diseñar los niveles utilizando el editor libre de mapas de tiles Tiled
